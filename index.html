Générateur de signatures électroniques

Remplis les champs, copie la signature et colle-la dans Gmail, Outlook ou Apple Mail.

Logo : le logo GAFA est déjà intégré dans la signature (visible partout, même sur mobile). Laisse le champ vide.
Coller : Gmail → Paramètres → Signature ; Outlook / Apple Mail → Préférences → Signature ; puis Ctrl/Cmd+V.
